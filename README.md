DESCRIPTION:
This project is a browser-based word search game built using HTML, CSS, and JavaScript.
The HTML provides the puzzle structure, CSS handles the visual layout and styling of the grid,
and JavaScript generates the word search, checks user selections, and tracks which words have been found.
The game highlights selected letters, marks completed words, and offers an interactive and responsive experience for players.

Features:
1. Dynamic Puzzle Generation

Creates the word search grid automatically.

Adjustable grid size.

Uses a 2D array structure for clean letter placement.

2. Random Word Placement Engine

Places words in multiple orientations:

Horizontal (forward/backward)

Vertical (up/down)

Diagonal (all directions)

Avoids placement conflicts.

Automatically fills empty spaces with random letters.

3. Interactive Letter Selection

Click-and-drag selection like a real word search.

Highlights the path in real-time.

Supports all directions.

4. Word Detection System

Matches selected letters with valid words.

Detects words in both forward and reverse order.

5. Word Highlighting

Highlights found words in the grid.

Marks completed words on the word list.

Prevents re-selecting finished words.

6. Word List Panel

Displays all target words for the puzzle.

Auto-updates when words are found.

7. Timer System

Stopwatch starts when the game begins.

Stops after all words are found.

8. Scoring System

Rewards points for each found word.

Easily expandable for leaderboard or difficulty modes.

9. Sound Effects

Plays sound when selecting letters.

Plays a distinct sound when a word is found.

10. Restart / Replay Button

Resets:

Grid

Words

Timer

Score

Highlights

Generates a fresh puzzle instantly.

11. Easy to Play Intro Screen

"Click anywhere to begin" intro.

Smooth fade-out transition into gameplay.

12. Themed UI & Animations

Hover and highlight effects.

Clean and responsive layout.

13. Full Dynamic DOM Construction

Grid and word list generated entirely in JavaScript.

Event listeners automatically added.

Separation of data, logic, and UI.

How to Play:
Click anywhere to start the game.

Click and drag across letters to form a word.

Release to check your selection.

If it matches:

The word is highlighted.

The word list updates.

Score increases.

Find all words to win!
